# Lesson 2 — Character Animation (Level 2)

## Goal
Students build their own animated character sequence using the animation library.

## Requirements for Level 2
- run movement and eye behavior asynchronously
- add blink loop (500 ms blank time)
- add at least one wink
- choose a TTS voice
- pre-generate phrases and play from cache
- include fidget movement (random, 0.05 s steps, 1/4 speed or slower)
- use camera gestures: nod, shake, look up/left/down/right
- use horn (normal)

## Outcome
Create a complete 15-30 second performance sequence.
The sequence should feel like a character scene, not isolated tests.

## Suggested structure
1) intro line + nod
2) talking segment with blink + fidget
3) reaction segment (look + shake)
4) ending move (drift/horn/pose)
